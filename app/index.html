A Study Of SASS Mixins
That May Or May Not Be Useful

Circle-Crop Mixin

The Circle-Crop mixin will take any desired image and crop it into a circle, perfect for those new circle avatars everyone seems to love these days. How does it work you ask? Choose your favorite picture. Inside HTML make an img tag and nest it within a div tag with a class name. In CSS call the class name and include the mixin circle-crop [@include circle-crop] Now you’re the proud owner of a new circle cropped image. See below for nerd notes.

@mixin circle-crop ($width, $img-width, $margin-top, $margin-left) { margin: 20px auto; width: $width; height: $width; overflow: hidden; border-radius: 100%; img { width: $img-width; margin: $margin-top 0 0 $margin-left; } } .body { @include circle-crop (100px, 600px, -30px, -70px); }

Color-Wash Mixin

The Color-Wash mixin will take any desired image and put a color-wash of your choosing over that image when hovered over. How does it work you ask? Choose your favorite picture and color. Inside HTML use an img tag and a div tag then nest those within a div with a class name. In the CSS call the class name and include the mixin color-wash [@include color-wash] Now you’re the proud owner of a new image with a color wash (Oh and, I threw in my circle-crop mixin just for fun). See below for nerd notes.

@mixin color-wash ($img-width, $img-height, $color) { margin: 50px auto; width: $pic-width; height: $pic-height; position: relative; img { width: $img-width; height: $img-height; z-index: 10; } .color { width: $img-width; height: $img-height; -webkit-transition: all 1s ease; position: absolute; top: 0; z-index: 20; } .color:hover { box-shadow: inset 0 -200px 0 rgba($color, 0.50); } } .body { @include color-wash (180px, 180px, #662134); @include circle-crop (100px, 180px, 0, -2px); }

The Hover-Over-Text mixin will take any desired image and put text with the color of your choosing over that image when hovered over. How does it work you ask? Choose your favorite picture and color. Inside HTML use an img tag and a div tag then nest those within a div with a class name. In the CSS call the class name and include the mixin hover-over-text [@hover-over-text] Now you’re the proud owner of a new image with text that appears when hovered over(oh and, I threw in my circle-crop mixin just for fun..one more time.). See below for nerd notes.

@mixin hover-over-text ($width, $img-width, $text-color) { margin: 50px auto; width: $width; height: $width; position: relative; background: #FFF; overflow: hidden; img { width: $image-width; z-index: 10; position: absolute; margin: -35px 0 0 -19px; } .text { line-height: $hvr-width - 50px; text-align: center; color: rgba($text-color, 0); font-size: $width - 65px; z-index: 20; position: relative; -webkit-transition: all 1s ease; } .text:hover { color: rgba($text-color, 1); } } .body{ @include hover-over-text (100px, 241px, #DCD611); @include circle-crop (100px, 170px, -8px, -15px) }
